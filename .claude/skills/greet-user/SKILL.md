# greet-user

这是一个**最小可运行的 Skill 示例**，用来演示 SKILL.md 的结构。

## 何时触发

- 用户明确说"问候我"、"打个招呼"、"say hi"
- 用户在新对话开始时想看 skill 是否生效

## 操作步骤

1. 用 PowerShell 取本机时间：`(Get-Date).Hour`
2. 根据小时数选问候语：
   - 5–11 → "早上好"
   - 12–17 → "下午好"
   - 18–22 → "晚上好"
   - 其他 → "夜深了，注意休息"
3. 拼接上用户邮箱的 local-part（@ 前面的部分）作为称呼。
4. 用一句话输出，例如：`早上好，tzysz1997！`

## 输出格式

只输出一行问候语，不要附加解释。

## 关键知识点（教学注释）

- Skill 是**按需加载**的。SKILL.md 的 frontmatter（name + description）会被列在系统提示里，但**正文只有触发时才进入上下文**，所以可以放心写得详细。
- description 字段决定模型何时调用，写得越具体越准。"包含触发关键词"和"反例"都有用。
- Skill 文件夹可以放任意辅助资源（脚本、模板、参考数据），用相对路径在 SKILL.md 里引用即可。
